# ParserVK
## Описание:
>Бот ищет пересечение групп пользователей с вашими 

### Бот состоит из
    1) inputParser.txt
    2) ParserVK.py
    3) parsVK.json
    4) result.txt


## Как использовать 
___
 - открыть файл **inputParser.txt** заполнить параметры поиска 
 - запустить **ParserVK.py**
 - открыть файл с результатами **result.txt**
